Detailed Description

Allows to update the firmware of a device.

To update the device firmware, call CollectFirmwareUpdateInformation() with a *.guf file and the device to update, select one of the firmware updates in the list, then pass it to UpdateDevice() to exectue the update.

To observe the update progress, pass a FirmwareUpdateProgressObserver to the UpdateDevice(). This is optional, but very helpful, since firmware updates can take several minutes.

Note
The DeviceDescriptor passed to CollectFirmwareUpdateInformation() and UpdateDevice() will be opened in Exclusive mode. Therefore, if the Device is opened in any other application, both methods will fail with an InternalErrorException.

Updates a given device by using a given firmware update information.

This is a blocking call, i.e. it only returns once the update is done. Depending on the device, the update may take several minutes. To watch the progress during the update, pass a FirmwareUpdateProgressObserver.

Parameters
[in]deviceThe device to update.
[in]updateInformationThe firmware update information to update the device.
[in]progressObserverThe progress observer to observe the update process. It is optional.
[in]deviceResetDiscoveryTimeoutTime to wait for a device to reboot during the update.
Note
The DeviceDescriptor passed to this function will be invalid after the update. Update the DeviceManager/Interface and retrieve a new DeviceDescriptor.
Since
1.0
1.2 Added deviceResetDiscoveryTimeout parameter.

InvalidArgumentExceptionOne of the submitted arguments is outside the valid range or is not supported.
InternalErrorExceptionAn internal error has occurred.
