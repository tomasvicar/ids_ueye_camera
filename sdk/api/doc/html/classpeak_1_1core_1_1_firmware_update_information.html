Detailed Description

Represents a single firmware update information.

This class allows to query information from the *.guf file about a single firmware update information.

See GenICam FWUpdate Standard.

std::string peak::core::FirmwareUpdateInformation::VersionExtractionPattern ( ) const
inline

Regular expression to extract the device version from the DeviceFirmwareVersion node.

The first matched group is used as result. This is needed for devices which encode more information than just the firmware version inside the DeviceFirmwareVersion node. The default value is: ^(.*)$
