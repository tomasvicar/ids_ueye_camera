Detailed Description

Represents a GenTL System module.

This class allows to query information about the GenTL System module and to enumerate its InterfaceDescriptors, which allow to open the corresponding Interface.

RawInformation peak::core::System::Info ( int32_t  infoCommand) const
inline

Delivers information based on the given GenTL info command.

Parameters
[in]infoCommandThe GenTL info command.

This function can be used to query information going beyond the predefined info functions, based on the GenTL info commands of the corresponding module.

Example (error handling is omitted):

#include <peak/thirdparty/GenTL.h>
auto info = object->Info(GenTL::XX_INFO_XX);
// Cast depending on the delivered data type
if (info.dataType == GenTL::INFO_DATATYPE_UINT64)
{
uint64_t uint64Var = *reinterpret_cast<const uint64_t*>(info.data.data());
// Do something with the information
}
else if (infoDataTypeVar == GenTL::INFO_DATATYPE_INT64)
{
...
}
...

Raw information according to the passed info command

void peak::core::System::UpdateInterfaces ( Timeout  timeout_ms)
inline

Searches for interfaces.

This function triggers an update of the internal interface list. The callbacks registered on the system will be triggered if an interface is found or lost.

Parameters
