Detailed Description

Allows to access the physical Device.

Access the NodeMap and Port of the physical device. This class doesn't have any additional functionality, but helps separate the local virtual device proxy from that actual physical hardware device. I.e. the NodeMap of this RemoteDevice is created from the XML file retrieved from the physical device, while the NodeMap of the proxy Device is created by the ProducerLibrary (CTI).
